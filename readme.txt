Custom Search Results helps you put the important results first on searches, shows what users are searching for on your site.

== Description ==

Get insight into what your users are searching for on your website and customize the results they see.
Do you have an important accouncement you want to show up at the top of your search results?  Is there a specific product you want to promote?  
Make sure your users get to see it when they search your site.

Major features include:

* Add Custom Search Results
* Order Custom Search Results
* Historical Search Statistics 
* Configure which post types will be searched

== Installation ==
= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Custom Search Results'
3. Activate Custom Search Results from your Plugins page.
4. Visit 'Custom Search Results' > 'Custom Post Types' 
5. Configure the posts types you want to be searchable
6. Visit 'Custom Search Results' > 'Custom Results' 
7. Configure the custom searches results you want to be displayed
8. Add Posts/Pages/Products... to custom searches from the 'Custom Search Results'-widget on the edit page of each post type.
